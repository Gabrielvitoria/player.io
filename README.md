##Player.IO

A media center thats run on your browser. 
Made using just **JavaScript**.

---

- Node.js
- Socket.io
- File Access API
- HTML5 Audio and Video
- jQuery Mobile 
- Jade Template Engine

---
 
To test application local:

- Instal socket.io and express.js

```bash
$ npm install socket.io express
```

- Instal Jade

```bash
$ npm install jade
```

- Run app.js

```bash
$ node app.js
```
